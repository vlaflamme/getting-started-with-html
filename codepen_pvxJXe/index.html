The Basics

Web - a collection of HTML documents and other content (e.g. plain text, HTML, images, videos, music, etc.

HTML - Hypertext Markup Language
- Text content - what you see
- Markup - what it looks like
- References to other documents (videos, etc.)
- Links to other pages - connects multiple pages together
Major Pieces
- The User Computer contains a program called a Browser that is used to interpret the HTML page and display it on the user's screen.

- The Internet is the world's largest computer network.

- HTTP protocol is the main protocol used on the web; this enables the user computer to send and receive HTTP Requests with a server.

- Servers are computers that host the files that make up the Web. The server finds the appropriate HTML document and sends it back to the user's computer, where the web Browser interprets and dispays the page.

- HTML Markup is made up of tags.

- HTML ignores white space; you have to code line breaks, etc. to display the text the way you want it.

- An Element refers to everything within a set of opening and closing tags.

- An Attribute is a property of an HTML tag (e.g. HREF is an attribute of the anchor tag).

- Tags can be inline or block; inline does not add any width or height; block tags surround the content in an invisible box, which adds height and width.

- HTML document structure contains a heading (typically the title of the page); document type; an opening and closing "html" tag which surrounds the main body of the HTML content; and the body, which contains the main content of the page.

Common Tags
- b - makes text bold
- em - makes text italic
- u - makes text underlined
- a - anchor - used for making links; contains attributes, e.g. HREF.
- img - image - used to include images on the page; does not have a closing tag
- br - inserts a line break; can have multiple; does not have a closing tag
- p - paragraph - block tag
